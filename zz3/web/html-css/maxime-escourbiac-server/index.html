Accueil

Bienvenue sur mon site personnel. Ce site a pour but de remplacer l'ancienne version crée avec i'Web. Vous y trouverez notamment les liens vers mes repository SVN de mes 3 années passées au sein de l'ISIMA. De plus, j'ai mis en place un serveur d'applications qui contient plusieurs outils du génie logiciel comme Hudson. Je peux le mettre à disposition pour les personnes souhaitant s'initier à ce genre de software.

Hudson disponible


Désormais, Hudson est disponible à l'adresse suivante. les fonctionnalités avec Maven n'ont pas été encore intégrées. Au niveau des jdk installés sur le serveur, les versions 1.6 et 1.7 d'oracle sont disponibles. Pour toute demande d'hébergement de projet, n'hésitez pas à m'envoyer un mail.

Mise en place des repository SVN.

Tous les cours et tous les tp de zz1,zz2 et zz3 sont disponibles via svn. Pour obtenir directement un fichier spécifique, vous pouvez directement y accéder via un navigateur web.

Java-Sci.

Java-sci est un framework scientifique en langage Java. La version actuelle est la 0.1. Cette version implementera les premiers éléments de base (vecteurs, matrices, complexes, générateurs de nombres aléatoires). L'implementation d'algorithme de théorie des graphes est prévue pour la version 0.2.
